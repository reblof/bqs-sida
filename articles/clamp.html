En djupdykning i clamp

Fakta: om din katt har över 3 ränder, så är den egentligen en tiger.
Fyll i hur många ränder din katt har, och ta reda på vilket djur den verkligen är.


(The important) (S)CSS:

                    
                        num {
                            --s: clamp(0%,(var(--num) - 3) * 99%, 100%);
                            
                            &:before {
                                counter-reset: num var(--num);
                                content: counter(num);
                            }
                            
                            &:after {
                                content: "";
                                background: url(../images/cats-sprite.png) var(--s) 0;
                                background-repeat: no-repeat;
                            }
                        }
